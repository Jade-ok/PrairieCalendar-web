🫐 PrairieCalendar

A Chrome extension that extracts PrairieTest schedules and allows you to export them to your Google Calendar or as an .ics file with ease. Built with ❤️ by Team BlueberryPie (Jena & Jade).

✨ What It Does

  • Smart Extraction: Parses exam titles, dates, times, and locations directly from the us.prairietest.com webpage.
  • Selective Exporting: Clean checkbox UI allows users to select specific events to export.
  • Instant .ics Generation: Converts your selected schedules into a valid iCalendar file locally.
  • Google Calendar Integration: Export your selected schedules directly to Google Calendar with one click. Events are added instantly via the Google Calendar API with built-in duplicate detection.

🔒 Permissions Explained

PrairieCalendar only requests the minimum permissions necessary to function. There is no external database; all processing is done locally to protect student data.

  • storage Used to store parsed reservation data locally on your device via Chrome's storage.local API.
  • identity Used to securely authenticate with Google via OAuth 2.0 if you choose to export to Google Calendar.
  • host permissions We strictly limit data extraction to the PrairieTest domain (https://us.prairietest.com/*) and communication to Google APIs (https://www.googleapis.com/*).
